\section{Dijkstra's Algorithm}

Cho do thi co huong co trong so $G = (V, E)$ voi $n$ dinh va $m$ canh, moi canh $(u, v)$ co trong so $w(u, v) \geq 0$.
Cho dinh nguon $s$, tim duong di ngan nhat tu $s$ toi tat ca cac dinh con lai. Goi $d[v]$ la do dai duong di ngan nhat tu $s$ toi $v$.

Cong thuc relax (noi long) la co so cua thuat toan:
\[
  d[v] = \min\bigl(d[v],\; d[u] + w(u, v)\bigr)
\]
Neu duong di qua $u$ roi toi $v$ ngan hon duong di hien tai toi $v$, ta cap nhat $d[v]$.

\subsection{Algorithm}

Thuat toan Dijkstra hoat dong theo nguyen tac tham lam (greedy): tai moi buoc, chon dinh $u$ chua xu ly co $d[u]$ nho nhat,
danh dau $u$ la da xu ly (settled), roi relax tat ca canh ke $(u, v)$. Dung priority queue (hang doi uu tien) de chon $u$
hieu qua trong $O(\log V)$. Dieu kien: trong so canh phai khong am ($w \geq 0$).

\subsection{Complexity}

Voi priority queue (binary heap), do phuc tap cua Dijkstra la $O((V + E) \log V)$:
moi dinh duoc push/pop dung mot lan ($O(V \log V)$), moi canh duoc relax dung mot lan voi thao tac decrease-key ($O(E \log V)$).
Neu dung Fibonacci heap, do phuc tap giam xuong $O(V \log V + E)$.
Luu y: Dijkstra chi dung khi trong so canh khong am. Neu co canh am, dung Bellman-Ford ($O(VE)$) hoac SPFA.

\subsection{Implementation}

\begin{lstlisting}[language=Python]
import heapq
from math import inf

def dijkstra(adj, src):
    n = len(adj)
    dist = [inf] * n
    dist[src] = 0
    pq = [(0, src)]

    while pq:
        d_u, u = heapq.heappop(pq)
        if d_u > dist[u]:
            continue
        for v, w in adj[u]:
            if dist[u] + w < dist[v]:
                dist[v] = dist[u] + w
                heapq.heappush(pq, (dist[v], v))

    return dist
\end{lstlisting}
