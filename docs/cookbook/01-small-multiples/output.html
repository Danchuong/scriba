Example 01 — Small multiples output

Expected HTML mà Scriba compiler emit cho input.md. Figure duy nhất, static, không JS controller.

Notes on what you're seeing

  • No JavaScript for this figure. Static figures không cần step controller. Chỉ có KaTeX auto-render (inherited từ Scriba TeX plugin) để render $n!$ trong caption.
  • figure có attributes id="fig-state-space", data-scriba-kind="figure", data-scriba-src="input.md:4". Consumer có thể \ref{fig:state-space} ở prose → auto-numbered link.
  • Mỗi panel group <g class="sm-panel"> có data-scriba-src chỉ về line trong input.md mà panel đó được khai báo. Dev tool inspect → nhảy về source. Production build có thể strip attr này để giảm size.
  • Bubble r (radius) được compute từ log(value) × scale: 24 → r=24, 720 → r=48, 362880 → r=82. Tác giả không set r thủ công.
  • Panel thứ 3 dùng color=danger → compile xuống D2 style.fill: "#fde2e2", style.stroke: "#dc2626". Semantic mapping từ color name sang Wong palette.
