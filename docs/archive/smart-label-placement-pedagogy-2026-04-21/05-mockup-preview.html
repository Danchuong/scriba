Smart-label v2.0.0 — mockup before/after

Mô phỏng kết quả khi áp dụng ruleset mới (R-01..R-30). Scene: 3 ô DP dp[0..2] với 3 transition arrows (good / info / muted). Tham chiếu: 00-synthesis.md §4.

Case 1 · Arrow annotations ở trạng thái natural (không displacement)

v0.10.0 Current engine
  • info pill contrast ≈ 1.95:1 (WCAG fail)
  • muted pill contrast ≈ 1.56:1 (WCAG fail)
  • info/muted giống nhau dưới CVD deuteranopia
  • pill overlap arc (natural y = mid − 4 px)
  • path token trùng màu info token
v2.0.0 Proposed — R-01, R-12, R-13, R-25
  • R-12: info opacity 0.45→0.7 · muted 0.30→0.6 → contrast ≥ 3:1
  • R-13: muted dash "1,3" trên arrow path (CVD-safe)
  • R-01: pill y = mid − pill_h/2 − 4 (không chạm arc)
  • R-25: path token đổi sang #6d28d9 (nếu dùng)
  • R-23: pill border opacity 0.3→0.7

Case 2 · Collision forces displacement → leader line behaviour

v0.10.0 Current engine
  • Leader threshold hard-coded 30 px (W-3)
  • Leader endpoint = pill center (cuts through text)
  • Mọi token đều emit leader → noise
  • Info leader vô nghĩa (pill ngay sát arc)
v2.0.0 Proposed — R-07, R-08, R-22, R-27
  • R-22: side_hint suy từ dst − src + obstacle → pill né sang trái
  • R-27: good/info/muted/path không emit leader
  • R-07: threshold = max(pill_h, 20), không còn magic 30
  • R-08: warn leader dừng ở pill perimeter, không cắt vào text
  • R-13: warn dash "3,2" trên arrow path (không chỉ leader)

Case 3 · Pill che mất target cell (v0.12.0 preview)

v0.10.0 Current engine
  • Registry chỉ thấy pills → target cell invisible
  • Pill nudged DOWN phủ lên giá trị 16 của dp[1]
  • Học sinh mất thông tin quan trọng nhất
v2.1.0-rc Proposed — R-02, R-18
  • R-02: target cell AABB pre-registered với kind="target"
  • R-18: registry thấy non-pill marks → nudge bỏ qua vùng cell
  • Pill nằm phía trên arc, dp[1] value vẫn đọc được
  • Cần resolve_obstacle_boxes() API (M cost)

Tóm tắt visible changes

  • Pill contrast tăng rõ rệt cho info/muted (WCAG AA passed).
  • Muted arrows có dotted dash trên path → distinguishable khi CVD/print grayscale.
  • Leader lines biến mất khỏi good/info/muted/path → ít noise, chỉ warn/error mới leader.
  • Arc labels lift cao hơn (pill_h/2 + 4) → không chạm stroke.
  • Pill direction tự suy từ arrow vector + obstacle → né thông minh hơn, không cần author đánh dấu side.
  • Target cells được bảo vệ khỏi occlusion (v0.12.0).
